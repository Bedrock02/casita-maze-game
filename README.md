# La Casita Maze Game

React + TypeScript maze game with endless procedural levels.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Controls

- Arrow keys to move
- Reach the pink casita to advance to the next level

## Features

- Starts at level 1 with an easy maze
- Infinite progression (maze size increases every level)
- Per-level elapsed timer
- Concert-inspired pixel avatar with direction-aware profile and walking feet animation
- Sugar-cane styled maze walls and pink casita exit goal
